📝 Blog Frontend

The WASM frontend is built with Yew and bundled by trunk. This file is a placeholder; the real app lives in blog-wasm/.

Build & run

# 1. Install trunk (one-time)
cargo install trunk

# 2. Build or serve the Yew app
cd blog-wasm
trunk serve --port 8000   # dev server (blog-server occupies 8080)
# or
trunk build --release     # production output → blog-wasm/dist/

# 3. Open in browser
open http://localhost:8000

Make sure blog-server is running on port 8080 before opening the frontend — the Yew app talks to http://localhost:8080.
